# Mondly USA Offer

A generated GitHub Pages site for USA visitors promoting Mondly with short daily lessons, speech practice, and conversation-style learning.

## What this project does

This repository builds a static affiliate site for Mondly. The generator creates a homepage, hub pages, long-tail pages, `robots.txt`, `sitemap.xml`, `llms.txt`, and GitHub Pages-ready output.

## Live site

- Home: [Mondly USA Offer](https://brightlane.github.io/mondly.com/)
- Review: [Mondly Review](https://brightlane.github.io/mondly.com/review/)
- FAQ: [Mondly FAQ](https://brightlane.github.io/mondly.com/faq/)
- Alternatives: [Mondly Alternatives](https://brightlane.github.io/mondly.com/alternatives/)

## Project structure

- `build.py` — Generates the site files.
- `index.html` — Home page generated into the repo root.
- `review/`, `faq/`, `alternatives/`, `guide/`, `best/`, `speaking/`, `travel/`, `beginner/`, `daily/` — Hub pages and generated page folders.
- `robots.txt` — Crawling instructions.
- `sitemap.xml` — URL list for search engines.
- `llms.txt` — Short project summary for LLM-friendly discovery.
- `.github/workflows/pages.yml` — GitHub Actions workflow for build and deploy.

## How it works

The site is generated by `build.py` and written directly into the repository root. The GitHub Actions workflow then commits the generated files and deploys them with GitHub Pages.

## Local build

```bash
python build.py
```

## Deployment

The site is deployed through GitHub Pages using GitHub Actions. The workflow builds the files, commits them to the repository, and publishes the static site.
